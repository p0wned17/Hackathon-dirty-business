## Порядок работы с проектом

### Прежде, чем начать


### 1. Запуск сервиса

Запустите сервис
```bash
make run_app
```

### 3. Нагрузочное тестирование

В проекте реализовано нагрузочное тестирование с помощью фреймворка locust


Запустите тестирование
```bash
make locust_test
```
